# MathsTests
MathsTests is an Open-source app which can help you factorizing polynomials.

## Supported:
- Trinomail factoring;
- Ruffini's rule;
- Fraction calculator.

### Extras:
- Material design (AppCompat & support library);
- Easter egg :-)
